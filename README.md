本项目是基于 n8n-nodes-feishu-lite 之上进行的二次开发，感谢原作者的开源贡献。在原有功能基础上，增加了更多实用的节点类型和功能支持。

与原项目最大的差别是添加了 Markdown/HTML 内容转换为文档块

参考：https://docs.n8n.io/integrations/community-nodes/installation/
